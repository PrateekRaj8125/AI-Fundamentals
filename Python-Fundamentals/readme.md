# Python Fundamentals 🐍

This folder contains Python programs that demonstrate **core programming concepts and data structures**. It is designed to build a strong foundation in Python by covering essential topics like control flow, collections, and built-in modules.

These programs help beginners understand **basic syntax**, **operations**, and **data handling in Python**

---

## 📚 Topics Covered

- Control flow statements (`if`, `if-else`, `elif`)
- Looping constructs (`for`, `while`)
- Tuple operations and manipulation
- Array operations using Python array module
- Set operations (`union`, `intersection`, `difference`)
- Dictionary operations and nested dictionaries
