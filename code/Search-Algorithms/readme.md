# Search Algorithms

This folder contains implementations of **fundamental AI search algorithms**, focusing on graph traversal and problem-solving techniques.

---

## 📚 Topics Covered

- Graph representation (Adjacency List)
- Breadth-First Search (BFS)
- Uniform Cost Search (UCS)
- State space exploration
- Cost-based pathfinding
