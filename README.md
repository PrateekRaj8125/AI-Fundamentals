# 🤖 AI Fundamentals

This repository contains a **comprehensive collection of Artificial Intelligence programs**, systematically organized from **basic Python concepts to advanced AI algorithms**.

It is designed to serve as a **complete learning path for AI Fundamentals**, covering:

- Python basics for AI
- Search algorithms
- Problem-solving techniques
- Heuristic and optimization methods
- Evolutionary algorithms

Each folder represents a **progressive level of learning**, making it ideal for:

- 📘 Academic lab work
- 💻 Practice and revision
- 🎯 Interview preparation

---

## 📂 Repository Structure

```text
    AI-Fundamentals/
    │
    ├── Python-Fundamentals/
    ├── Search-Algorithms-BFS-UCS/
    ├── AI-Problem-Solving-and-Search/
    ├── Heuristic-Search-and-Game-Theory/
    ├── Evolutionary-Algorithms-and-CSP/
```

## 📚 Folder Overview

| Level | Folder Name | Concepts Covered | Description |
