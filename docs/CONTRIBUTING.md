# 🤝 Contributing to AI-Fundamentals

Thank you for considering contributing! 🎉
Your contributions help improve this repository as a learning resource for Artificial Intelligence.

---

## 📌 Ways to Contribute

You can contribute in multiple ways:

- 🐞 Fix bugs or incorrect logic
- 💡 Suggest new AI problems or improvements
- 📝 Improve documentation (README, explanations)
- 💻 Add new AI algorithms or notebooks
- ⚡ Optimize existing implementations

## 🛠 Development Setup

1. Fork the repository
2. Clone your fork:

    ```bash
        git clone https://github.com/PrateekRaj8125/AI-Fundamentals.git
        cd AI-Fundamentals
    ```

3. Run Python Programs

    ```bash
        python filename.py
    ```

4. Run Jupyter Notebooks

    ```bash
        jupyter notebook
    ```

---

## ✅ Contribution Guidelines

- Follow clean coding practices:
  - Proper indentation
  - Meaningful variable names
  - Clear comments
- Ensure:
  - Code runs without errors
  - Output is correct
  - Logic is clear and efficient

---

## 🐞 Reporting Issues

Use GitHub Issues and include:

- Description of the issue
- Steps to reproduce
- Expected vs actual output
- Screenshots/logs (if needed)
