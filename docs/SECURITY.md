# Security Policy

## 📢 Reporting a Vulnerability

We take security and code reliability seriously.

If you discover a vulnerability, bug, or unexpected behavior in this repository, please follow responsible disclosure:

---

### 🚫 Do NOT

- Publicly disclose the issue before it is reviewed and resolved

---

### ✅ Do

1. Create a [GitHub Security Advisory]()
    OR
2. Report via email: <prateek.raj.indt@gmail.com>

---

## 📄 What to Include in Your Report

Please include:

- Clear description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Potential impact
- Suggested fix (if available)

---

## 🔔 What to Expect

- ⏱ Acknowledgment within **48 hours**
- 🔍 Investigation within **7 days**
- 🛠 Fix release if confirmed
- 🙌 Credit (optional)

---

## 🎯 Scope

This repository includes:

- Python programs
- AI algorithms (search, heuristic, evolutionary)
- Jupyter notebooks

Potential issues may include:

- Logical errors in algorithms
- Incorrect outputs
- Inefficient implementations
- Notebook execution issues

---

✅ Responsible reporting helps maintain a reliable and high-quality AI learning resource.
